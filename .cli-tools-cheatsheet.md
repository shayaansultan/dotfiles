# Modern CLI Tools Cheatsheet

Quick reference for modern terminal tools. All are faster, prettier alternatives to built-ins.

---

## File Finding: `fd` (replaces `find`)

```bash
fd pattern              # Find files matching pattern
fd "\.py$"              # Find Python files (regex)
fd -H pattern           # Include hidden files
fd -t f pattern         # Files only (not directories)
fd -t d pattern         # Directories only
fd -e ts                # Find by extension (.ts files)
fd pattern --exec cmd   # Run command on each result
```

## Content Search: `rg` (replaces `grep`)

```bash
rg pattern              # Search recursively
rg pattern -t py        # Search only Python files
rg pattern -t ts -t tsx # Search TS/TSX files
rg -l pattern           # List filenames only (no content)
rg -i pattern           # Case insensitive
rg -w pattern           # Whole word match
rg -C 3 pattern         # Show 3 lines context
rg "TODO|FIXME"         # Multiple patterns (regex)
rg -g "*.py" pattern    # Glob filter
```

## Fuzzy Finder: `fzf`

```bash
fzf                     # Fuzzy find from stdin
Ctrl+R                  # Fuzzy search shell history (if configured)
fd | fzf                # Find file interactively
rg -l pattern | fzf     # Search results → pick one
cat file | fzf          # Pick a line from file

# Open selected file in editor
vim $(fd | fzf)
code $(fd -e ts | fzf)
```

## File Viewing: `bat` (replaces `cat`)

```bash
bat file.py             # View with syntax highlighting
bat -p file.py          # Plain (no line numbers/headers)
bat -l python file      # Force language
bat -n file             # Line numbers only (no header)
bat --diff file         # Show git diff
bat -A file             # Show invisible chars (tabs, etc)
```

## Directory Listing: `eza` (replaces `ls`)

```bash
eza                     # Basic listing
eza -la                 # Long format + hidden
eza -la --git           # Show git status per file
eza --tree              # Tree view
eza --tree -L 2         # Tree, 2 levels deep
eza -l --sort=modified  # Sort by modified time
```

## Smart Navigation: `zoxide` (replaces `cd`)

```bash
z dirname               # Jump to most-used matching dir
z maven                 # Jump to maven (learns from usage)
z back                  # Jump to backend if you go there often
zi                      # Interactive mode with fzf
zoxide query maven      # See what it would match
```

**Setup required** (add to ~/.zshrc):
```bash
eval "$(zoxide init zsh)"
```

## Git TUI: `lazygit`

```bash
lazygit                 # Open in current repo
```

**Key bindings in lazygit:**
- `space` - Stage/unstage file
- `c` - Commit
- `p` - Push
- `P` - Pull
- `b` - Branch menu
- `?` - Help
- `q` - Quit

## Git Diffs: `delta`

Delta works automatically when configured in git. Add to `~/.gitconfig`:

```ini
[core]
    pager = delta

[interactive]
    diffFilter = delta --color-only

[delta]
    navigate = true
    line-numbers = true
    side-by-side = false
```

Then `git diff`, `git show`, `git log -p` all use delta automatically.

## JSON Processing: `jq`

```bash
jq '.'                  # Pretty print JSON
jq '.key'               # Extract key
jq '.data[]'            # Iterate array
jq '.[] | .name'        # Get name from each item
jq -r '.key'            # Raw output (no quotes)
jq 'keys'               # List all keys

# Real examples
curl api/endpoint | jq '.data'
cat file.json | jq '.users[] | {name, email}'
```

## Safe Delete: `trash` (replaces `rm`)

```bash
trash file.txt          # Move to trash (recoverable!)
trash *.log             # Trash multiple files
```

Recover from Finder → Trash, or `Cmd+Z` right after.

## Simple Man Pages: `tldr`

```bash
tldr tar                # Practical tar examples
tldr git-rebase         # Common git rebase patterns
tldr fd                 # Examples for any command
tldr --update           # Update local cache
```

## System Monitor: `btop` (replaces `top`/`htop`)

```bash
btop                    # Launch monitor
```

**Key bindings:**
- `h` - Help
- `e` - Toggle tree view
- `f` - Filter processes
- `k` - Kill process
- `q` - Quit

## Prompt: `starship`

Shows git branch, node/python version, command duration, etc.

**Setup** (add to ~/.zshrc):
```bash
eval "$(starship init zsh)"
```

Configure at `~/.config/starship.toml`

## Quick Aliases (add to ~/.zshrc)

```bash
# Modern replacements
alias ls='eza'
alias ll='eza -la --git'
alias cat='bat -p'
alias find='fd'
alias grep='rg'
alias top='btop'
alias rm='trash'  # Careful: changes rm behavior!

# Useful combos
alias fzp='fzf --preview "bat --color=always {}"'
alias lg='lazygit'
```
